# Selenium Automation Framework (Python)

This project demonstrates a scalable Selenium automation framework using Python, Pytest, and Allure Reporting.

## Features

* Page Object Model (POM)
* Pytest-based test execution
* Allure reporting integration
* WebDriver Manager for automatic driver setup

## Tech Stack

* Python
* Selenium
* Pytest
* Allure Reports

## How to Run Tests

```Powershell
py -m pytest
```

## Generate Allure Report

```Powershell
py -m pytest --alluredir=reports
allure serve reports
```

## Test Case Included

* Valid login test using demo site
